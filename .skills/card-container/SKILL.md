# CardContainer

**Import:** `import { CardContainer } from "@penbot/core";`

## Usage

A bordered container for displaying demos/examples that fits the docs theme.

```svelte
<CardContainer class="flex flex-wrap gap-4">
  <Button variant="brand">Demo Button</Button>
</CardContainer>
```

Apply Tailwind classes via the `class` prop for layout.
