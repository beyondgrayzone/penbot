# Textarea

**Import:** `import { Textarea, Label } from "@penbot/core";`

## Usage

```svelte
<Label for="bio">Your bio</Label>
<Textarea id="bio" name="bio" placeholder="Tell us about yourself" />
```

Supports standard HTML textarea attributes (`id`, `name`, `placeholder`, `disabled`, etc.).
